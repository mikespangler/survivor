## Scoring Breakdown

### Points Per Episode
- **Question Points**: 5-10 points per correct answer
- **Wager Points**: 1-25 points (can be negative if wrong)
- **Retention Points**: 2 points per active castaway

### Example Team Episode Calculation
```
Episode 8 for a sample team:
- Question 1 (5 pts) - Correct ✓ = +5
- Question 2 (5 pts) - Correct ✓ = +5
- Question 3 (WAGER 15 pts) - Wrong ✗ = -15
- Active Castaways (2) = 2 × 2 = +4
= Episode Total: -1 points
```
